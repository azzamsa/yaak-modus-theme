# Modus Themes for Yaak

Highly accessible themes for [Yaak](https://yaak.app/), conforming with
the highest standard for color contrast between background and foreground values
([WCAG AAA](https://www.w3.org/WAI/WCAG21/Understanding/contrast-enhanced.html)). Port of the original [Modus Themes](https://protesilaos.com/emacs/modus-themes).
